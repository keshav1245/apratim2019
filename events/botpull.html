Bot Pull








DIMENSIONS OF THE ROBOT

  • Width : Maximum 250mm

  • Length : Maximum 250mm

  • Height : Maximum 200mm


RULES:

  • General rules of the game:

    • A robot must win 2 out of 3 trials to win the match

  • In game rules:

    • If any part of a robot crosses the centreline, or any part comes off a robot, that robot loses that round.

    • If, after 1 minute, no robot has crossed the centreline, the round is to be considered draw.

    • Robot cannot be touched during the match.

    • NOTE: EVENT WILL BE ORGANISED INDOOR



  • Regarding robot:

    • The robots can be controlled with wireless / wired remote controller .

    • Weight: Robot must not exceed 2 kg.

    • The robot must be started manually with a start button. The start button should have the word written near the button. .

    • The robot must be powered by a power source such as a battery fixed on the robot.

    • There is no limitation on the number of motors and rpm of motors to be used as long as it satisfy above dimensions.

    • NOTE: Robots that cause deliberate interference with other robots or damage to the field will be disqualified.

NOTE: Humans that cause deliberate interference with robots or damage the field will be disqualified
